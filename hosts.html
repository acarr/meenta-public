Who can be a host?

Any commercial or academic instrument owner verified by Meenta can become a Host.

How does Meenta ensure that only qualified users contact the Hosts?

While anyone can search the Meenta platform, only Users with a validated payment method can request a booking from a Host.

How does the Host get paid?

Meenta will pay you with electronic wire or check. Your payment is sent automatically within 48 hours of completion of the project.

What is the service fee Meenta charges Hosts?

Meenta takes an 8% Hosts reservation fee on every booking.

How does Meenta deliver the sample data?

Through the secure Meenta Cloud at no charge to the Host or the Scientists.

API & SDK Documentation

Meenta offers a REST API for user, partner and equipment integrations. To access the API you need an active account with API access enabled. Use the account API options to generate an API token. This API implements standard OAuth2 with variable rate limiting per endpoint grouping.
